

/****** Regras para variavel******/

1 - Só podemos utilizar números, letras maisúsculas, minúsculas, em um único caractere especial, que é o sublinhado ou em inglês underline_

2 - O primeiro caractere deve ser sempre uma letra ou um underline_____

3 - Não são permitidos espaços em branco ou outros caracteres especiais,

4 - Não podemos utilizar palavras reservadas, ou seja, algumas palavras que identificam algumas coisas da linguagem de programação;

=========Tipos de de variáveis no JAVA===================================

variáveis ( tipos de dados primitivos e seus intervalos)

O conceito de tipos primitivos refere-se aos tipos de dados mais básicos fornecidos por uma 
linguagem de programação. Eles são chamados de "primitivos" porque servem como blocos fundamentais 
para a construção de informações mais complexas e não são compostos por outros tipos.

- byte - Números entre -128 e 127
        byte idade = 43;
- short - Números entre -32768 e 32767
        short numeroCadastro = 134569;
- int - Números entre - 2147483648 e 2147483647
        int numeroCasa = 610;
- long - Números entre - -9223372036854775808 e 9223372036854775807
        long 
- float  - Números com até 15 casas decimais
        float preco = 342.96f;
- double - Números com até 15 casas decimais
        double total = 1352960.25;
- boolean - Armazena valores verdadeiro ou falso
        boolean itemVisivel =  false; // true(verdadeiro false(falso))
- char - Armazena apenas um caracter

       /****** Atribuição direta de dados ******/

A atribuição direta de dados é um conceito comum em programação 
e significa atribuir um valor diretamente a uma variável.

A atribuição direta de dados é um conceito comum em programação e significa atribuir um valor 
diretamente a uma variável.

Outra forma de atribuir um valor a uma variável é utilizar o comando de leitura do teclado. 
No entanto, o valor que será inserido na variável irá depender do que o usuário irá digitar, enquanto 
a atribuição direta de dados, quem define o conteúdo a ser atribuido direto na variável é o programador.



/ * Exemplo */

public class Main{
  public static void main(String[]args){
    String nome;
    String sobrenome;
    int idade;
    float peso;

    nome = "Bruna";
    sobrenome = "De Sá"
    idade = 42;
    peso = 88.5f;

    System.out.println(nome);
    System.out.println(sobrenome);
    System.out.println(idade);
    System.out.println(peso);
  }
}
